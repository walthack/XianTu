# 六朝清羽记·草原鹰帜

## 定位

这是 Scenario Mod 系统的首个真实验收包，覆盖《六朝清羽记》第一集开篇的草原战局。它不是整部小说改编，也不要求玩家扮演程宗扬。

- 模式：Strict
- 规模：6 章、10 个事件
- 玩家身份：独立异界来客或战场流落者
- 结束点：草原军团冲突告一段落，玩家选择进入六朝世界的下一条道路

## 设计原则

1. 程宗扬、王哲、月霜等作为正典 NPC 存在。
2. 玩家可以改变局部人物命运和事件结果，但不凭个人力量抹除军团级冲突。
3. 章节摘要只描述目标，不替玩家决定行动。
4. 生死根通过 `rules.contentAccess` 锁定为程宗扬唯一持有；独立玩家和其他 NPC 不可学习或获得。
4. 成人与强迫性桥段不纳入本 Mod。
5. 不复制小说正文，只保留摘要化事实和剧情锚点。

## 已知限制

- XianTu 的部分底层玩法仍默认使用通用修仙境界。Mod 通过 `world.specialRules` 要求使用六朝九境语汇，但无法完全替换所有数值系统。
- Scenario Mod v1 只保存实体摘要，不能表达完整军制、经济或复杂分支图。
- 当前 Mod 用 flags 表达里程碑；具体如何达成由 LLM 和玩家行动共同决定。

## 校验

```bash
npm run mod:validate -- mods/liuchao-qingyu-prologue/liuchao.qingyu.grassland.json
```

导入后应验证：Strict 模式不调用世界生成、开场位于“月牙平原山丘”、初始章节为“月牙平原”、后续章节在提示上下文中不可见、存档重载后剧情进度仍然存在。
