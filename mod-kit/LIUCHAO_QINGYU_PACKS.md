这三份 JSON 根据 `/Volumes/botsvault/06_material` 中的《六朝清羽记》剧情大纲、人物卡、背景分析和 `六朝-XianTu-catalog-v1.md` 提炼，不复制小说正文。

2. `lcq-act2-merchant-rise.json`：第12至24章，受制身份到商人立足。
3. `lcq-act3-southsea.json`：约第25至58章，南荒、海王神殿与碧鲮海湾。

原“势力初成”草稿混入《六朝燕歌行》的刘建、定陶王与汉宫政治线，已移除。现有概要不足以把第59至80章写成高保真独立 Strict 包；在核对更详细章节材料前不补造剧情。

## 人物关系边界

Scenario Mod v1 已映射原生关系系统：

- 宗派、军队、国家、家族和组织归属使用 `canon.characters[].affiliations`。
- `canon.playerRelationships` 写入 `社交.关系`，提供 NPC 对玩家的初始关系、好感度与记忆。
- `canon.relationships` 写入 `社交.关系矩阵.edges`，表示 NPC 之间的师徒、亲友、盟敌和上下级关系。
- `relatedCharacterIds` 仍只表示人物参与某事件，不代表关系类型。
- 初始关系不永久锁死；好感度和关系强度会继续随玩家行为与剧情事件变化。

## 核心正典

- 程宗扬是正典 NPC；玩家为进入同一时期的独立角色。
- 生死根仅属于程宗扬。
- 王哲保持太乙真宗掌教和左武军师帅的分类身份。
- 乐明珠属于光明观堂；小紫只保留与巫宗、毒宗相关的复杂背景；凝羽保持侍卫身份；阿姬曼·芭娜保持天竺来客与舞姬身份。
- 未在素材中明确的阶段配角会在包内 README 中标为可玩性补充，不冒充原作正典人物。
